There are 4 mitigation methods that can mitigating the attack technique called Bypass User Account Control. Their names are as follows:
* Update Software 
* Audit 
* User Account Control 
* Privileged Account Management 

The following content describes information about mitigation methods. The format of each mitigation method's information is as follows:
Mitigation method name: ***
Mitigation method description: ***
These respectively represent the name of the mitigation method and a detailed description of how it works.

Mitigation method name: Update Software
Mitigation method description: Perform regular software updates to mitigate exploitation risk.

Mitigation method name: Audit
Mitigation method description: Perform audits or scans of systems, permissions, insecure software, insecure configurations, etc. to identify potential weaknesses.

Mitigation method name: User Account Control
Mitigation method description: Configure Windows User Account Control to mitigate risk of adversaries obtaining elevated process access.

Mitigation method name: Privileged Account Management
Mitigation method description: Manage the creation, modification, use, and permissions associated to privileged accounts, including SYSTEM and root.
